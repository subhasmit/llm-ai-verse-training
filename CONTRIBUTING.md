Thank you for your interest in contributing to the LLM AI Verse Training repository! We appreciate your support in making this a valuable resource for the community.

### Reporting Issues

If you find a bug or have a suggestion:

1. Check if the issue already exists in [GitHub Issues](https://github.com/subhasmit/llm-ai-verse-training/issues)
2. If not, create a new issue with a clear title and description
3. Include relevant details such as:
   - Episode number (if applicable)
   - Steps to reproduce the issue
   - Expected vs actual behavior
   - Your environment (OS, Python version, etc.)

### Submitting Pull Requests

1. **Fork the repository** and create your branch from `main`
2. **Make your changes** with clear, descriptive commit messages
3. **Test your code** to ensure it works correctly
4. **Update documentation** if needed
5. **Submit a pull request** with a clear description of your changes

### Code Style Guidelines

- Follow PEP 8 style guidelines for Python code
- Use meaningful variable and function names
- Add comments for complex logic
- Include docstrings for functions and classes
